# 🛒 React Shopping App

A modern React-based shopping application with product listing, cart management, authentication flow, and multi-step checkout.

---

## 🚀 Features

- 🛍️ Product Listing (API आधारित)
- 🔍 Search with Debouncing
- 🎯 Category Filter & Price Sorting
- 🛒 Cart System (Add, Remove, Update Quantity)
- 🔐 Authentication (Login / Logout using Context API)
- 🧾 Multi-Step Checkout Form
- 📦 Cart Summary in Final Step
- 🚫 Protected Routes
- ⚡ Responsive UI

---

## 🧠 Tech Stack

- React.js
- React Router
- Context API
- useReducer
- Custom Hooks (useDebounce)
- CSS
